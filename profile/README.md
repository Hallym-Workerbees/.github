# 🐝 Hallym Workerbees

> Building HiveWiki — an AI-powered knowledge structuring platform.

## Introduction

Workerbees는 한림대학교 소프트웨어학부 학생들로 구성된 개발 팀입니다.

우리는 단순한 웹 애플리케이션 제작을 넘어,  
AI 기반 지식 구조화와 Cloud Native 환경에서의 서비스 운영 경험을 목표로 프로젝트를 진행하고 있습니다.

현재 진행 중인 주요 프로젝트는 **HiveWiki**입니다.

## Team Philosophy

벌집(Hive)은 수많은 Worker Bee들의 협업으로 유지됩니다.

우리 팀은 다음과 같은 가치를 중요하게 생각합니다.

- 협업 중심 개발 문화
- 문서 기반 커뮤니케이션
- 자동화 우선 접근

- 지속 가능한 아키텍처
- 책임 있는 역할 분담

## About HiveWiki

HiveWiki는 비정형 데이터를 자동으로 구조화하여  
위키 형태의 지식 시스템으로 변환하는 AI 기반 플랫폼입니다.

단순 요약 서비스가 아니라:

- 문서 간 관계를 연결하고
- 계층형 Wiki 구조를 생성하며
- 최신 정보를 지속적으로 반영하고
- 사용자에게 검색 및 질의응답 기능을 제공하는

지식 허브 구축을 목표로 합니다.

### 주요 대상 데이터

- 학교 공지사항
- 학사 정보
- 행사 정보
- 커뮤니티 게시글
- 외부 웹 문서

## Key Features

### 📚 Automatic Wiki Generation

비정형 문서를 분석하여 계층형 Wiki 자동 생성

### 🔗 Knowledge Relationship Mapping

문서 간 관계 추론 및 링크 연결

### 📰 Continuous Data Collection

RSS 및 웹 크롤링 기반 데이터 수집 파이프라인

### ☁️ Cloud Native Deployment

GitOps 기반 배포 및 운영 환경 구성
